# Tic-Tac-Toe React App

A modern Tic-Tac-Toe game built with React and Vite.

## Features

- Interactive React-based Tic-Tac-Toe game
- Player name editing functionality
- Clean, modern UI with CSS styling
- Game board and turn management

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd 07-tic-tac-toe-starting-project
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal)

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the app for production
- `npm run preview` - Preview the production build
- `npm run lint` - Run ESLint to check for code issues
